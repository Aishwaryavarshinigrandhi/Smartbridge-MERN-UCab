# 🚖 UCab – MERN Stack Cab Booking System

## 📌 Project Overview

UCab is a full-stack cab booking web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application enables users to register, log in securely, book cab rides, calculate fares, and view their booking history through a responsive and user-friendly interface.

This project demonstrates the practical implementation of full-stack web development concepts, including REST APIs, authentication, database management, and frontend-backend integration.

---

## 🚀 Features

- 👤 User Registration
- 🔐 Secure User Login with JWT Authentication
- 🚖 Cab Booking System
- 💰 Automatic Fare Calculation

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/aishwaryavarshinigrandhi/Smartbridge-MERN-UCab.git
```

### 2. Navigate to the Project

```bash
cd UCab-MERN-Cab-Booking-System
```

### 3. Install Backend Dependencies

```bash
cd server
npm install
```

### 4. Install Frontend Dependencies

```bash
cd ../client
npm install
```

### 5. Configure Environment Variables

Create a `.env` file inside the **server** folder and add the required MongoDB connection string and JWT secret.

Example:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### 6. Run the Backend

```bash
cd server
npm start
```

### 7. Run the Frontend

```bash
cd client
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
